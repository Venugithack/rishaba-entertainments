# Rishaba Entertainments

Official website for **Rishaba Entertainments** — a Chennai-based film production and content house producing Tamil cinema and premium OTT web series.

A static, responsive single-page site (HTML / CSS / vanilla JS) hosted on GitHub Pages.

## Structure

```
index.html        # page markup
css/styles.css    # cinematic dark + gold theme, fully responsive
js/main.js        # nav, scroll progress, reveals, scrollspy, contact form
assets/           # optimized logo files
```

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8000   # then visit http://localhost:8000
```

## Deployment

Hosted with **GitHub Pages** from the `main` branch (root). Any push to `main` updates the live site.
